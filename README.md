# Full-Stack Employee Management System

This is a **Full-Stack Employee Management System** built with:
- **Frontend**: React (Vite) + Bootstrap CSS  
- **Backend**: Spring Boot (Java)  
- **Database**: MySQL  

It allows users to:
- Add a new employee
- Update employee details
- Delete an employee
